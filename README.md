# Oasis_Infobyte_Internship_Data_Analytics 
Data Analytics internship projects for Oasis Infobyte. Features 6 comprehensive tasks including Retail Sales EDA, Customer Segmentation, House Price Prediction, Wine Quality Classification, and App Market Sentiment Analysis.

##  Project Overview
This repository serves as a comprehensive showcase of the data analytics and machine learning tasks completed during my internship at **Oasis Infobyte**. The project is structured into two levels of difficulty, demonstrating a progression from exploratory data processing to predictive modeling and sentiment engineering.

### Objectives:
* Extract actionable insights from diverse datasets (Retail, Real Estate, and Mobile Apps).
* Apply advanced statistical cleaning and data preparation techniques.
* Develop robust machine learning models for classification and regression.
* Communicate complex data findings through professional, course-aligned visualizations.
